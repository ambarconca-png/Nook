# Codex: Start here

This repository already contains a functional UI prototype.

Your first tasks:
1. Install dependencies and run `npm run build`.
2. Fix any issues without redesigning the app.
3. Commit the stable baseline.
4. Add Supabase auth and persistence incrementally.
5. Keep the German copy and agreed visual identity.

Product rules:
- This is a personal home, not a generic productivity tool.
- Capture first, organize later.
- Dashboard only shows today's relevant information.
- Areas may contain direct todos and optional projects.
- No streak pressure for routines.
- Cycle predictions must be labeled as estimates.
- No classic emoji icons.
- Brand name is always lowercase: nook.
